About Me

SpendWise - Personal Finance Tracker

SpendWise is a full-stack expense tracking web application built with Django. It features a custom authentication system, CRUD functionality for managing expenses, and automated generation of reports by date, month, and year. The backend uses Django ORM with SQLite in development and PostgreSQL for production deployment. The frontend leverages Bootstrap for a clean and responsive interface. Deployed on Render, the project demonstrates end-to-end development skills including database modeling, user session management, static file handling, and cloud deployment.
